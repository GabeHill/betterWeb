The slideshow is made with a div of class slideshow-container which then holds a number of divs, one for each image, and each having the classes slide and fade.

Each div will have a div with class numbertext to hold what number of image it is. Then an image tag with class slide-img. There will also be a following div of class text for the optional image caption.

Two a tags at the bottom, one class prev and the other next. Each has an onClick="plusSlides()". Previous will have a -1 passed in and next will have a 1 passed in.

Outside of the slideshow container theres wil be another div with class dot-holder with a span for each dot each having class dot and an onClick="currentSlide()" with the corresponding slide number passed in.
